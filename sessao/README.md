# Sessão 🎞️❤️

Este é o app (React + Vite). A documentação completa — visão geral, features, setup e deploy — está no **[README principal do repositório](../README.md)**.

## Atalhos

```bash
npm install      # instalar dependências
npm run dev      # servidor de desenvolvimento (http://localhost:5173)
npm run build    # build de produção (gera dist/)
npm run preview  # servir o build localmente
```

Crie um `.env` com as chaves do TMDB (veja o [README principal](../README.md#-rodando-localmente)):

```bash
VITE_TMDB_READ_TOKEN=...
VITE_TMDB_API_KEY=...
```
